# Como inicializar el proyecto
Primer paso: instalar las dependencias
```
npm install 
```

Segundo paso: inicializar servidor JSON

```
npx json-server --watch  data/db.json --port 8000
```


Tercer paso: inicializar aplicación

```
npm start
```
